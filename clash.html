使用步骤

  1. 复制订阅地址
    返回首页，点击 Clash 订阅卡片的「复制链接」按钮，将订阅地址复制到剪贴板。
  2. 打开客户端，导入订阅
    打开 Clash 客户端，找到「配置」或「Profiles」页面，在订阅地址输入框中粘贴刚才复制的链接，点击「导入」或「Download」。
  3. 选择配置文件
    导入成功后，在配置列表中点击选中刚导入的配置文件，使其变为激活状态。
  4. 开启系统代理
    在客户端主界面，打开「系统代理」或「System Proxy」开关。选择代理模式为「规则」或「Rule」模式。
  5. 选择节点
    在「代理」或「Proxies」页面，可以选择「AUTO」自动选择延迟最低的节点，也可以手动选择特定节点。

常见问题

导入失败怎么办？
检查订阅地址是否完整复制，确认网络可以正常访问 GitHub Pages。如果仍然失败，尝试下载文件后本地导入。
节点无法连接？
免费节点可能随时失效，请等待每日自动更新。也可以在代理页面切换到其他节点尝试。
配置文件说明
本订阅提供完整的 Clash/Mihomo 配置文件，包含代理节点、代理组（PROXY 选择组、AUTO 自动测速组）和路由规则，可直接导入使用。
